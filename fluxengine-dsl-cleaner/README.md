# fluxengine-dsl-cleaner
登録済みのDSLを削除するためのプロジェクト。
主としてCIで使用するが、ここでは単体の使用方法を説明する。

# Dependency
使用言語：java
ビルドツール：gradle

# Version
Fluxengine 1.0.11

# SetUp

1. 本プロジェクトをチェックアウトする

2. fluxengine-dsl-cleaner/conf/dsl.properties を適宜編集する

# Usage

gradleでプロジェクトを実行する  
```./gradlew run```
